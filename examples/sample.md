# Sample Document

This document demonstrates **file input** rendering with the CXY markdown library.

## Features

The markdown parser supports a wide range of features:

- **Bold** and *italic* text
- Inline `code` formatting
- [Links](https://cxy-lang.org)
- Blockquotes
- Tables
- And much more!

### Code Examples

Here's a Rust code block:

```rust
fn main() {
    println!("Hello from Rust!");
}
```

And here's some CXY code:

```cxy
func greet(name: String) {
    println(f"Hello, {name}!")
}
```
